# ServiceNow Incident Categories

## Standard Categories

| Category | Subcategories |
|----------|--------------|
| Hardware | Desktop, Laptop, Monitor, Peripheral, Printer |
| Software | Application, Operating System, License, Update |
| Network | Connectivity, VPN, DNS, Firewall, Wireless |
| Email | Outlook, Exchange, Shared Mailbox, Distribution List |
| Database | SQL Server, Oracle, Performance, Backup |
| Security | Access, Malware, Phishing, Account Lockout |

## Impact Levels

| Impact | Description |
|--------|-------------|
| 1 - High | Enterprise-wide or critical business function affected |
| 2 - Medium | Department or group affected |
| 3 - Low | Individual user affected |

## Urgency Levels

| Urgency | Description |
|---------|-------------|
| 1 - High | Business-critical function unavailable, no workaround |
| 2 - Medium | Business function degraded, workaround available |
| 3 - Low | Minimal business impact |
